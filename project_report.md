# Project Report

## Main Obstacles

The main challenge was organizing the quiz so that questions could easily be modified without changing the game logic. Separating the questions from the main program solved this problem.

Another challenge was ensuring that invalid user input did not crash the program. Input validation was added to repeatedly ask for a valid answer.

## Main Results

The project successfully implements an educational quiz game about Finnish history.

Features include:

- Multiple-choice questions
- Score tracking
- Input validation
- Immediate feedback
- Final results summary

The code is divided into multiple files to improve readability and maintenance.

## Use of Large Language Models

Large language models were used to assist with:

- Planning the project structure
- Generating Python source code
- Designing quiz questions
- Writing project documentation
- Improving code organization and readability

All generated content was reviewed and tested before inclusion in the project.
